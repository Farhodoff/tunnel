# Loyiha maqsadi

Tunelio — lokal veb serverlarni internetga xavfsiz va qulay tarzda chiqarishga mo‘ljallangan tunneling ilovasi. U ngrok yoki localtunnel kabi vositalarga yengil alternativ bo‘lib, webhooks va masofaviy testlar uchun tezkor ulanish, real vaqt monitoringi hamda API token orqali himoyani ta’minlaydi.

# Stacklar

- Python 3.8+
- FastAPI
- Uvicorn
- WebSockets
- aiohttp
- Redis (ixtiyoriy)
- Prometheus
- Docker va Docker Compose
- SSL/TLS (Cryptography)
- Pytest
- MkDocs
